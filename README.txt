This ZIP contains a single-file static HTML MVP for an Investment-like platform.

Usage:
1. Unzip and open index.html in a browser.
2. This is a frontend-only mock. No backend, no real payments.
3. To accept payments or store users, integrate with a backend and a payment gateway (Stripe/PayPal).

Warning: Do NOT accept real funds using this file as-is. Consult legal and security experts before launching.
